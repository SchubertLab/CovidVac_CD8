# CovidVaccineStudy
## Summary
Data analysis of a vaccine study for Sars-CoV-2 
- 4 Donors
- Pre + post (10 days) vaccine injection
- Unstimulated / stimulated (with Covid mix) T cells
- Transcriptome and TCR information (e.g. reverse phenotyping)
